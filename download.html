Shomer — Email Guardian

Install the Shomer Companion App

The browser extension protects you on its own. The optional companion app adds sender verification (SPF/DKIM) and Windows Sandbox isolation for suspicious links — features that need to run outside the browser.

!Do you need this?
Shomer's core protection — AI email analysis, link scanning, and file scanning — works without the companion app, on any operating system. Install this only if you want the advanced Windows-only features.
1Download
⬇️ Download for Windows
ShomerAgentSetup.exe · ~40 MB · Windows 10/11
🍎 macOS & Linux: the companion app is Windows-only. All other features work in your browser.
2Run the installer
Double-click ShomerAgentSetup.exe. It installs the agent and connects your Gmail once (you pick your account). No system Python or admin knowledge needed.
⚠️ SmartScreen notice: Windows may show "Windows protected your PC" for new installers. Click More info → Run anyway. (We're working on a signed installer to remove this.)
3Done — reload Gmail
Refresh your Gmail tab. You'll now see SPF/DKIM sender-authentication results and can open suspicious links in Windows Sandbox.
What the companion app adds
✓ Real SPF/DKIM sender authentication from email headers
✓ Windows Sandbox — open risky links in a throwaway VM
✓ Inline attachment scanning without leaving Gmail
Questions
Is my data safe?
Your Gmail connection is encrypted on your PC (Windows DPAPI) and never sent to our servers. The app only reads message headers, not your inbox. See our Privacy Policy.
Why isn't this in the Chrome Web Store?
Chrome extensions can't run native features like Windows Sandbox. The Web Store installs the extension; this installs the companion that powers the advanced features.
How do I uninstall it?
Windows Settings → Apps → Shomer Email Guardian → Uninstall. This removes the agent and your stored Gmail connection.
